## 🔄 Mantenimiento

### Política de Actualización

- **README.md:** Actualizar con cada release
- **SISTEMA_100_FUNCIONAL.md:** Actualizar con cambios mayores
- **HOJA_DE_RUTA.md:** Revisión mensual
- **SIGUIENTE_PASOS.md:** Actualización semanal
- **Docs técnicas:** Actualizar al resolver problemas

### Criterios para Archivar

Un documento se archiva cuando:
- ❌ La información está obsoleta
- ❌ El problema descrito ya está resuelto
- ❌ Existe un documento más actualizado
- ❌ Es duplicado de otro documento
- ❌ No aporta valor al flujo actual

### Proceso de Limpieza

1. Analizar todos los `.md` cada sprint
2. Identificar duplicados y obsoletos
3. Mover a `.archive_YYYYMMDD/`
4. Actualizar `DOCS_SUMMARY.md`
5. Actualizar referencias en documentos activos
